# Enhanced Underwater Image Restoration

Sea-Thru Preprocessing + FUnIE-GAN + Real-ESRGAN

## Overview

This project enhances underwater images using a multi-stage computer vision pipeline combining:

- Sea-Thru inspired preprocessing
- FUnIE-GAN enhancement
- Real-ESRGAN super-resolution

The system improves underwater visibility, color correction, and image sharpness.

---

## Pipeline

Input Image
→ Sea-Thru Preprocessing
→ FUnIE-GAN Enhancement
→ Real-ESRGAN Upscaling
→ Final Output

---

## Features

- Physics-guided underwater correction
- GAN-based image enhancement
- Super-resolution sharpening
- Supports seen and unseen underwater datasets

---

## Tech Stack

- Python
- PyTorch
- OpenCV
- FUnIE-GAN
